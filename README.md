## Проект по теории сложностей

Заключается в вычислении запрещённых миноров (см. теорема Робертсона-Сеймура) для свойства "при удалении любого ребра есть вершинное покрытие из k вершин". Запрещённые миноры лежат в папке results.

vcf = vertex covering forbidden, cvcf = connected vertex covering forbidden, avcf = almost vertex covering forbidden.

Для воспроизведения: нужно установить пакет nauty, библиотеку networkx, запустить ячейки из proj.ipynb.

Также имеется исходник теха к отсчету.
